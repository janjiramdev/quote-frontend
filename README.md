# merchant-web-react

## Description

This project is quote frontend web with React + TypeScript + Vite

## Prerequisite

- Node version 20.0.0 or higher

## Setup project

- Config env follow up .env.example
- Install NodeJS dependencies

```bash
$ npm install
```

## Develop project

```bash
# Start project on development mode
$ npm run dev

# Format code
$ npm run format

# Validate code
$ npm run lint
```

## Build and preview project

```bash
# Build project
$ npm run build

# Preview project
$ npm run preview
```
